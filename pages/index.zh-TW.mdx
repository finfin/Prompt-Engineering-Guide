# 提示工程指南

提示工程是一個新興領域，專門研究開發和優化提示，以便高效地使用語言模型（LM）進行各種應用和研究主題。提示工程技能有助於更好地理解大型語言模型（LLM）的能力和局限。

研究人員利用提示工程提高 LLM 在各種常見和複雜任務（例如回答問題和算術推理）上的能力。開發人員使用提示工程設計與 LLM 和其他工具交互的穩健且有效的提示技術。

提示工程不僅僅是設計和開發提示。它涵蓋了與 LLM 互動和開發的各種技能和技巧。與 LLM 交互、建立並理解 LLM 能力是一項重要技能。您可以使用提示工程來提高 LLM 的安全性並構建新功能，例如使用領域知識和外部工具擴展 LLM。

受到與 LLM 開發的高度關注，我們創建了這個全新的提示工程指南，其中包含了所有與提示工程相關的最新論文、學習指南、模型、講座、參考資料、新的 LLM 功能和工具。
