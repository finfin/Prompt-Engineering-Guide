# 簡介

提示工程是一門相對較新的學科，用於開發和優化提示，以便高效地利用語言模型（LM）進行各種應用和研究主題。提示工程技能有助於更好地了解大型語言模型（LLM）的能力和局限性。研究人員利用提示工程提高 LLM 在各種常見和複雜任務（如回答問題和算術推理）上的能力。開發人員使用提示工程設計強大且有效的提示，用以介接 LLM 和其他相關工具。

本指南介紹了提示的基本知識，以提供如何使用提示與 LLM 進行互動與引導的粗略概念。

除非另有說明，否則所有示例都使用 [OpenAI 的遊樂場](https://platform.openai.com/playground) 測試 `text-davinci-003`。該模型使用預設配置，即 `temperature=0.7` 和 `top-p=1`。
